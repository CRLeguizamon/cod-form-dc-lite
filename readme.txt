Transforma el botón "Agregar al carrito" en un formulario modal simplificado para pago contra entrega.

== Description ==

Modal COD Form Lite for woocommerce convierte el proceso de checkout tradicional de WooCommerce en una experiencia simplificada mediante un formulario modal. Ideal para tiendas que manejan pagos contra entrega (COD) y dropshippers.

= Características principales =

* **Formulario modal personalizable** - Reemplaza el botón "Agregar al carrito" con un modal de checkout
* **Compatible con productos simples y variables** - Funciona con todos los tipos de productos WooCommerce
* **Integración completa con WooCommerce** - Mantiene toda la funcionalidad nativa del carrito y pedidos
* **Envíos flexibles** - Soporta envíos gratuitos o métodos nativos de WooCommerce
* **Shortcodes incluidos** - Agrega botones de compra en cualquier página
* **Personalización completa** - Colores, textos, estilos y layout completamente customizables
* **Campos opcionales** - Habilita/deshabilita campos según tus necesidades
* **Ícono de carrito para menús** - Shortcode para mostrar ícono de carrito en navegación

= Funcionalidades incluidas =

* Formulario de checkout en modal
* Campos de información del cliente customizables
* Integración con sistema de envíos de WooCommerce
* Shortcode para páginas de destino: `[custom_add_to_cart_button id="123"]`
* Shortcode para ícono de carrito: `[cod_menu_cart]`
* Soporte para términos y condiciones
* Personalización de botones y estilos
* Configuración de país base automática

= Casos de uso =

* **Dropshipping** - Formulario optimizado para conversión rápida
* **Tiendas COD** - Perfecto para pagos contra entrega
* **Landing pages** - Usa shortcodes para crear páginas de venta
* **Checkout simplificado** - Reduce pasos en el proceso de compra

= Configuración =

1. Instala y activa WooCommerce
2. Instala y activa Modal COD Form
3. Ve a WooCommerce > Configuración > COD Form
4. Personaliza tu formulario según tus necesidades
5. ¡Listo para usar!

= Shortcodes disponibles =

**Botón de compra para landing pages:**
`[custom_add_to_cart_button id="123"]`

**Ícono de carrito para menús:**
`[cod_menu_cart color="#000000"]`

== Installation ==

= Instalación automática =

1. Ve a Plugins > Añadir nuevo en tu panel de WordPress
2. Busca "Modal COD Form WooCommerce Lite"
3. Instala y activa el plugin
4. Ve a WooCommerce > Configuración > COD Form para configurar

= Instalación manual =

1. Descarga el archivo ZIP del plugin
2. Ve a Plugins > Añadir nuevo > Subir plugin
3. Selecciona el archivo ZIP y haz clic en "Instalar ahora"
4. Activa el plugin
5. Ve a WooCommerce > Configuración > COD Form para configurar

= Requisitos =

* WordPress 5.0 o superior
* WooCommerce 3.0 o superior
* PHP 7.4 o superior

== Frequently Asked Questions ==

= ¿Funciona con todos los temas de WordPress? =

Sí, el plugin está diseñado para funcionar con cualquier tema que sea compatible con WooCommerce. Los estilos del modal son independientes del tema.

= ¿Puedo usar el plugin sin WooCommerce? =

No, este plugin requiere WooCommerce para funcionar correctamente ya que extiende su funcionalidad.

= ¿Funciona con productos variables? =

Sí, el plugin es completamente compatible con productos variables de WooCommerce. El modal mostrará las opciones de variación correspondientes.

= ¿Puedo personalizar el diseño del formulario? =

Completamente. Puedes personalizar colores, textos, campos, layout y mucho más desde la página de configuración.

= ¿Cómo obtengo el ID de un producto para los shortcodes? =

Ve a WooCommerce > Productos, selecciona el producto deseado. En la URL del navegador verás algo como `post=123`. El número `123` es el ID del producto.

= ¿El plugin afecta el SEO de mi sitio? =

No, el plugin no afecta el SEO. Solo modifica la experiencia de checkout manteniendo toda la estructura de WooCommerce intacta.

= ¿Puedo deshabilitar campos del formulario? =

Sí, en la configuración puedes habilitar/deshabilitar campos como email, dirección 2, notas del pedido, etc.
